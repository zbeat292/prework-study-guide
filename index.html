CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • A CSS rule contains a selector(ie:h2), property(ie:color), and value (ie:blue).
  • An external CSS style sheet is most commonly used because it allows developers to keep all their CSS rules in a seperate file, making design changes easier.

Git

  • git status: checks what branch we a currently on
  • git checkout -b *branch-name*: creates a new branch and switches to it
  • git push origin *branch name*: Transmits local branch to the repo.

JavaScript

  • A variable is a name container that allows us to store data in our code.
  • Control flow is the order in which a cpu executes code in a script.
